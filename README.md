# Car Plate Detection

A deep learning-based project for detecting car license plates using computer vision techniques — **without OCR**. This project focuses only on locating license plates in images or video streams.

---

## Overview

This project detects the presence and location of vehicle license plates using image processing and deep learning models. It does **not perform character recognition**; instead, it outputs bounding boxes around detected plates.

---

## Features

* License plate detection
* Supports both images
* Fast inference with optimized models
* Accurate bounding box predictions
* Easy to train and test on custom datasets

---

## Tech Stack

* Python
* OpenCV
* NumPy

---

## Model Details

* Model Type: CNN / Object Detection 
* Task: Bounding box detection only
---

## Dataset

* Custom dataset of car images with labeled license plates
